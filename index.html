Setup Database

Aplikasi ini membutuhkan Firebase Anda sendiri. Silakan tempelkan objek konfigurasi (firebaseConfig) dari Firebase Console Anda di bawah ini:

⚠️ SANGAT PENTING: Langkah Wajib di Firebase Jika Anda mendapati error "Could not reach Cloud Firestore backend", pastikan:
1. Buka Firestore Database di Firebase Console lalu klik tombol Create Database.
2. Ke menu Authentication > Sign-in method > Aktifkan Email/Password.
3. Di menu Firestore Database > tab Rules, ubah menjadi: rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
} Lalu klik Publish.
